## 老项目依据

### 视频库

入口和 UI：

- `src/Views/VideoWindow.xaml`
- `src/Views/VideoWindow.xaml.cs`
- `src/Controls/VideoLibraryWebViewControl.cs`
- `src/Resources/Web/video_library/`

业务服务：

- `src/Services/VideoLibraryService.cs`
- `src/Services/VideoLibraryAbstractions.cs`
- `src/Services/VideoPreviewService.cs`
- `src/Services/SubtitleService.cs`（只读，字幕模块已经迁移）

必须闭环的现有行为：

- 新增/扫描视频条目、刷新元数据。
- 收藏切换、备注、播放进度和完成状态。
- 专辑新增、重命名、删除、加入专辑、移出专辑。
- 标签添加、重命名、删除及批量标签操作。
- 删除库记录时不删本地文件。
- 删除本地文件时沿用老项目“移入回收站并删除库记录”的行为。
- 单项和批量操作沿用现有确认框文案。
- 播放、打开文件位置及播放器入口使用现有 React UI，不新增第二套入口。

### 远程视频中心

入口和 UI：

- `src/Views/RemoteVideoCenterWindow.xaml`
- `src/Views/RemoteVideoCenterWindow.xaml.cs`
- `src/ViewModels/RemoteVideoCenterViewModel.cs`

业务服务：

- `src/Services/RemoteVideoServices.cs`
- `src/Services/RemotePlaybackServices.cs`
- `src/Services/RemoteSiteAdapters.cs`
- `src/Services/VideoSourceResolver.cs`

必须闭环的现有行为：

- 单链接与批量链接解析。
- 解析结果详情、清晰度选择、复制链接、播放和下载动作。
- 下载任务进度、取消、失败信息和完成状态。
- 下载记录删除规则：下载中不能删除；完成记录按老项目处理本地文件和视频库。
- 播放记录列表和再次播放。
- 下载目录、工具路径、并发等老项目当前有效设置；废弃设置不要迁移。
- 诊断信息必须来自真实服务状态，不得填假数据。
- 复用已经完成的站点配置和受保护 Cookie，不得把 Cookie 放进普通 JSON、日志或前端列表。
